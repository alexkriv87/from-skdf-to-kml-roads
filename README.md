# from_skdf_to_kml_roads

Скрипт для выгрузки дорог из СКДФ (https://xn--d1aluo.xn--p1ai/map) по координатам двух углов прямоугольника с сохранением в KML.

## Возможности

| Функция | Описание |
|---------|----------|
| Парсинг координат | Яндекс.Карты (`55.972483, 36.911828`) и SAS.Планет (`N51°43'15.9790" E121°07'42.0984"`) |
| Загрузка дорог | По bounding box (прямоугольнику) |
| Характеристики | Категория, покрытие, полосы, скорость, владелец |
| Ширина | Краткий диапазон + детальные участки |
| Осевая нагрузка | Краткий диапазон + детальные участки |
| Километровые столбы | Только для федеральных дорог |
| Экспорт в KML | Цвета по категориям, папки, подпапки по владельцам и дорогам |

## Структура проекта

| Файл | Назначение |
|------|-----------|
| `config.py` | Настройки (цвета, категории, шаблон description) |
| `coord_utils.py` | Парсинг координат, построение bbox |
| `geometry_funcs.py` | Конвертация проекций (метры ↔ градусы) |
| `logger_config.py` | Логирование (консоль + файл) |
| `skdf_api.py` | Запросы к API СКДФ |
| `kml_exporter.py` | Экспорт в KML |
| `template_*.kml` | Шаблоны KML |
